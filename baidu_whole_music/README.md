1、抓取百度整个网站的音乐    
2、远程部署到服务器，实现分布式抓取    

###抓取思路:    
1、解析所有歌手所在页面（http://music.baidu.com/artist），抓取歌手姓名链接的url    
2、请求姓名链接的url，判断是否为多页数据    
3、若为单页，则抓取当前页面歌曲列表并写入到本地；若为多页，则执行翻页抓取    

关于翻页抓取：    
（1）解析姓名链接url，获得总页码数    
（2）做get请求（http://music.baidu.com/data/user/getsongs），params参数有start（歌曲起始位置，为保持与百度页面数据同步，start始终为25的倍数），ting_uid（歌手ID），order（可有可无，但是要是没有该参数，抓取的歌曲列表与百度页面显示的不同）    
（3）解析返回的json格式数据    
注：get请求每次返回25条数据
